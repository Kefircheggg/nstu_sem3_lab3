## Сборка и запуск (C++)

CMake.

### 1. Конфигурация

Создайте директорию для сборки и сконфигурируйте проект:

```bash
cmake -S . -B build
```

### 2. Компиляция

```bash
cmake --build build
```

### 3. Запуск

После успешной сборки в директории `build` появятся исполняемые файлы.

**Запуск демонстрационного приложения:**

```bash
./build/lab3
```

**Запуск тестов:**

```bash
./build/unit_tests
```

**Запуск бенчмарков:**

```bash
./build/benchmarks
```
