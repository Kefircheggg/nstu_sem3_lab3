# Лабораторная работа №3: Структуры данных

Этот проект реализует основные структуры данных на C++ (и Go) с возможностью сериализации/десериализации и бенчмаркинга.

## Структуры данных

Реализованы следующие структуры:
- Динамический массив (`Array`)
- Односвязный список (`SinglyLinkedList`)
- Двусвязный список (`DoublyLinkedList`)
- Стек (`Stack`)
- Очередь (`Queue`)
- Хеш-таблица (`HashTable`)
- Красно-черное дерево (`RBTree`)

## Требования

- C++17 совместимый компилятор (GCC, Clang, MSVC)
- CMake 3.10 или выше
- Git (для загрузки зависимостей)

## Сборка и запуск (C++)

Проект использует систему сборки CMake.

### 1. Конфигурация

Создайте директорию для сборки и сконфигурируйте проект:

```bash
cmake -S . -B build
```

### 2. Компиляция

Соберите проект:

```bash
cmake --build build
```

### 3. Запуск

После успешной сборки в директории `build` появятся исполняемые файлы.

**Запуск демонстрационного приложения:**

```bash
./build/lab3
```
Это приложение демонстрирует базовую работу со всеми реализованными структурами.

**Запуск тестов:**

```bash
./build/unit_tests
```
Или через CTest:
```bash
cd build
ctest
```

**Запуск бенчмарков:**

```bash
./build/benchmarks
```

## Версия на Go

В проекте также присутствует реализация на языке Go. Она находится в директории `go-version`.

Для работы с Go версией перейдите в соответствующую директорию:

```bash
cd go-version
```

Там вы найдете свой `README.md` и `Makefile` для сборки и запуска.
